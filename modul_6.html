Moduł 6 - Listy

Podstawy pracy z listami w Pythonie - tworzenie, indeksowanie, modyfikacja i operacje.

Wprowadzenie

Listy w Pythonie służą do przechowywania danych. Tworzymy je za pomocą nawiasów kwadratowych [ ].

Przykład:
lista = [1, 2, 3, 'Baba Jaga', 'patrzy']       # możemy też stworzyć pustą listę: x = []

W listach możemy przechowywać różne typy danych np. ciągi znaków, liczby, boole: (True, False)

Indeksowanie

WAŻNE!!! 
Indeks od lewej zaczyna się od zera, a od prawej od -1.Zapisujemy indeks w nawiasach kwadratowych.

Lista = [ 1,  2,  3,  'Baba Jaga',  'patrzy' ]
         [0] [1]  [2]      [3]           [4]
         [-5][-4] [-3]    [-2]          [-1]

Podsumowywując:

lista[0] -> 1
lista[1] -> 2
lista[-1] -> 'patrzy'
lista[4] -> 'patrzy'

Czyli lista[0] wyświetli nam jedynkę, lista[1] da nam dwójkę,
 ale ‘patrzy’ będzie miało zarówno [-1] jak i [4].

Wyświetlanie i iteracja

Wyświetlić listę możemy za pomocą polecenia:

print(lista)    # co wyświetli: [1, 2, 3, 'Baba Jaga', 'patrzy']

lub za pomocą pętli:

for k in lista:
    print(k)

Da nam to:
1
2
3
Baba Jaga
patrzy

Dodawanie elementów

Do listy możemy dodawać rzeczy używając polecenia .append:
lista.append('mam cię')   # dopisze to na końcu listy

UWAGA! Tym poleceniem dodajemy po jednej wartości naraz.

Przykład:

l = [1, 3, 4, 5]
l.append(6)
print(l)   # [1, 3, 4, 5, 6]

Możemy też dodać element w odpowiednie miejsce: .insert(indeks, wartość)

l.insert(1, 2)   # w indeks[1] da 2 i reszta przesunie się o 1
                 # l = [1, 2, 3, 4, 5, 6]

Usuwanie elementów

l.remove(1)   # usunie  wartości 1 z listy
l.pop()       # usuwa ostatni element 
l.pop(2) / del l[2]     # usuwa element o indeksie 2 (reszta się przesunie)
del l[2]      # usuwa element o indeksie 2 (reszta się przesunie)
del l[:]      # usuwa zawartość listy, ale nie samą listę
del l         # usuwa listę wraz z zawartością

Przydatne funkcje i operacje

Funkcja max i min przyda się do sprawdzania największej lub najmniejszej wartości:

liczby = [1, 2, 333, 4, 5, 6, 7, 88, 9]
x = max(liczby)
print(x)  # wyświetli 333

l.sort()    # posortuje listę rosnąco ([1,4,5,3,2] → → → [1,2,3,4,5])
l.reverse() # odwróci kolejność elementów ([1,4,5,3,2] → → → [2,3,5,4,1])

Długość listy sprawdzamy poleceniem len:

y = len(lista)
print(y)  # wypisze ilość elementów w liście ,nie ostatni indeks

Listy możemy do siebie dodawać (łączyć):

lista3 = lista1 + lista2

Możemy też zmieniać wartości:

lista[1] = 'nie wiem co'

dane = []   #definicja pustej listy o nazwie dane

imie=str(input('Podaj jak masz na imię : '))
