Moduł 7 - Krotki

Krotki wprowadzenie

Krotki są nie naruszalne. Nie można ich modyfikować. Przechowują różne typy zmiennych.
Tworzone są za pomocą nawiasów okrągłych ( ).

Przykład:

Krotka = (1, 2, 3, 'jajko')  # i po co są nam krotki?
                             # dzięki nim mamy pewność, że nigdy nie zostaną zmienione

Rozpakowywanie krotek:

Krotka = (1, 2, 3, 'jajko')
x = Krotka[0]
y = Krotka[1]
z = Krotka[2]
a = Krotka[3]

Usuwanie krotki

MOŻEMY USUNĄĆ KROTKĘ:
del Krotka

Krotki są przydatne do przechowywania stałych zestawów danych, np. współrzędnych, stałych konfiguracji lub wartości, które nie powinny ulec zmianie podczas działania programu.
