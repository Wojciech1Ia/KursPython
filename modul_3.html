Moduł 3 - Pętle

Jak powtarzać kod i kontrolować przepływ programu.

Wprowadzenie

Jeżeli nie chciałbyś pisać 100 razy tego samego to użyłbyś pętli.
Pętla to fragment kodu, który powtarza dany blok kodu.
Poznamy dwa rodzaje pętli: FOR i WHILE.

Pętla for

Używamy jej, gdy wiemy ile razy chcemy powtórzyć blok.

Składnia:
for nazwa in range(koniec) - np. powtórzy 3x

range(koniec) → wykona się koniec razy (od 0 do koniec-1)
range(początek, koniec)
range(początek, koniec, skok)

Dlaczego koniec-1?
Bo Python "liczy" od 0.

Przykłady pętli for

Jeśli chcemy napisać 'code' pięć razy to będzie to wyglądać nastepująco:

for x in range(5):
    print('code')

for x in range(1, 6):
    print('code')

for x in range(0, 10, 2):
    print('code')

Bez pętli:
print('code\n' * 5)

Pętla while

Pętla while działa dopóki warunek jest prawdziwy.

Przykład nieskończonej pętli:
x = 1
while x == 1:
    print("nieskończona pętla")

Aby przerwać:
x += 1
 Gdy dodamy x+=1 ( x = x+1, czyli, że x w kolejnej iteracji (pw kolejnym wejściu w 
pętlę) x zwiększy się o 1), dzięki czemu w drugim ‘sprawdzaniu warunku na początku pętli’ x 
będzie równe 2 i pętla nie zadziała.

Przykład działania pętli while

X = True
Y = 100

while X == True:
    print(Y)
    Y -= 1
    if Y < 0:
            X = False

# Wypisze liczby od 100 do 0.Gdy Y = 1 i x = True to kod wypisze 1, a y będzie wtedy równe 0 (bo 1-1=0) i 
z powrotem wejdzie w pętlę (x=True) i wypisze Y(0) i odejmie od Y 1 i 
dopiero wtedy warunek Y < 0 będzie prawdziwy (True) i x będzie False i pętla się skończy.

break i continue

break → przerywa całą pętlę
continue → pomija bieżącą iterację i przechodzi do następnej

Przykład:
while x < 100:
    x += 1
    if x == 50:

Nesting 

Nesting to zagnieżdżanie pętli, czyli innymi słowy pętla w pętli. 
Wygląda ono a następująco :  

for i in range(1, 11):
    for n in range(1, 11):
        print(n * i, end='\t')        #WAŻNE by w odpowiednich
    print(i)                          #miejscach dawać wcięcia
